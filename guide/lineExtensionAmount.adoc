|===
|Path |Description

|`cbc:LineExtensionAmount`
|**Net amount** +
The net amount for the invoice line.
|`cbc:LineExtensionAmount/@currencyID`
|**Net amount currency** +
Currency applied for cbc:LineExtensionAmount, must be the same as cbc:DocumentCurrencyCode.
|===
